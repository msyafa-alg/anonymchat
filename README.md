## ğŸ”§ **Installation**

### **Prerequisites**
- [Node.js](https://nodejs.org/) (v14 or higher)
- [Git](https://git-scm.com/)
- [Vercel CLI](https://vercel.com/cli) (optional, for local testing)

### **Local Setup**

1. **Clone the repository**
   ```bash
   git clone https://github.com/msyafa-alg/anonymchat.git
   cd anonymchat
   ```

2. **Install dependencies**
   ```bash
   npm install
   ```

3. **Run locally with Vercel**
   ```bash
   npm install -g vercel
   vercel dev
   ```

4. **Open in browser**
   ```
   http://localhost:3000
   ```

## ğŸ® **Usage**

### **For Users**
1. Visit the [live demo](https://anonymchat-bysyafa.vercel.app)
2. You'll automatically get a random username (e.g., `Anonym-1234`)
3. Type your message and press Enter or click send
4. Watch messages appear in real-time
5. Toggle dark mode with the moon/sun icon
6. Use emoji picker to add fun emojis

### **For Developers**
```javascript
// API Endpoints
const API_URL = '/api/index';

// Join chat
POST /api/index { action: 'join' }

// Send message
POST /api/index { 
  action: 'send', 
  userName: 'Anonym-1234', 
  message: 'Hello World' 
}

// Get new messages
GET /api/index?lastId=123

// Heartbeat (keep connection alive)
POST /api/index { 
  action: 'heartbeat', 
  userName: 'Anonym-1234' 
}

// Leave chat
POST /api/index { 
  action: 'leave', 
  userName: 'Anonym-1234' 
}
```
